# RetainIQ Dashboard

Two ways to view the analytics:

## 1. Interactive HTML dashboard (no install)
Open [`index.html`](index.html) in any browser, or host it free via **GitHub Pages**:
1. Repo **Settings → Pages → Source: `main` branch → `/dashboard` folder** (or move `index.html` to repo root / `/docs`).
2. Your dashboard goes live at `https://sahaj-chakka.github.io/RetainIQ/`.

It's a single self-contained file — all data is inlined, charts render via Chart.js.

## 2. Build it in Tableau Public / Power BI (recommended for a BI portfolio)
The `tableau_ready/` folder has tidy, pre-aggregated extracts so you can build the
same views in minutes and publish to **Tableau Public** (free) or **Power BI**, then
link the published workbook from the main README.

| File | Grain | Use for |
|------|-------|---------|
| `source_month_fact.csv` | booking_source × month | main fact table (revenue trend, source mix) |
| `monthly_revenue.csv` | month | revenue line / forecast overlay |
| `cancellation_by_source.csv` | booking_source | cancellation bar chart |
| `revenue_by_source.csv` | booking_source | revenue contribution |
| `model_benchmark.csv` | model | model comparison table |

### Suggested Tableau views
1. **KPI tiles** — total revenue, cancellation rate, avg booking value (from `source_month_fact`).
2. **Revenue trend** — line of `revenue` by `month`, color by `booking_source`.
3. **Cancellation by source** — horizontal bar of `cancellation_rate`.
4. **Revenue contribution** — bar of revenue % by source.
5. **Model leaderboard** — bar of `AUC` by `model`, highlight the max.

Arrange on one dashboard, add a title and the **"synthetic data"** disclaimer, publish to
Tableau Public, and paste the embed link into the repo README.

> ℹ️ All extracts are synthetic — generated by `src/data_generation/generate_synthetic_data.py`.
